## 简介
基于 ***DJI A Board*** 搭建的电控框架库，与大多数嵌入式框架一样分为 `Bsp`、`Module`、`Sys`、`App`层。

框架使用 全`C++` 进行组织，并严格面向对象 

## 使用
用户 ***无需更改*** App层以下的任何代码，只需要通过调用 / 编写 `Application` 即可实现逻辑
